## mccollstreet

### Setup oh-my-zsh

#### Install mccollstreet plugin
```zsh
pushd $ZSH/custom/plugins && \
  git clone git@github.com:johnlayton/mccollstreet.git mccollstreet && \
  popd || echo "I'm broken"
```
```zsh
plugins=(... mccollstreet)
```

### Setup other

```zsh
pushd $HOME && \
  git clone git@github.com:johnlayton/mccollstreet.git .mccollstreet && \
  popd || echo "I'm broken"
```

```zsh
source ~/.mccollstreet/mccollstreet.plugin.zsh
```
